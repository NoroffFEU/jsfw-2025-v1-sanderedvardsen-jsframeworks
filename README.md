# ShopNest

An online shop built with Next.js for the JavaScript Frameworks course at Noroff.

## Built With

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- React Hook Form + Zod

## Getting Started

1. Clone the repo
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the dev server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000)

## Features

- Browse products from the Noroff API
- Search/filter products by name
- View individual product details and reviews
- Add/remove items from a cart (persisted in localStorage)
- Checkout page with order confirmation
- Contact form with validation

## API

All product data is fetched from the [Noroff Online Shop API](https://docs.noroff.dev/docs/v2/basic/online-shop).
